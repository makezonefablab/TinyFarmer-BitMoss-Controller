소개
--------------
농작물의 환경데이터를 수집하는 비트모스(BitMoss TM)는 다양한 센서와 연결하고 부가 기능의 액세서리 보드를 연결하여 확장 가능한 구성으로 이루어졌습니다.
전용 AP (TinyFarmer AP)와의 연결로 쉽게 데이터를 모아 중앙 센터로 전송합니다.
